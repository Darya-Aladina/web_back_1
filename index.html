  3. Использование ping

    С помощью команды ping на учебном сервере узнали IP-адрес веб-сервера kubsu.ru.
    Основная задача утилиты ping― определение работоспособности узла, сайта или сервера в интернете. Команда ping посылает небольшие пакеты на сервер, который нужно проверить. Через некоторое время от сервера приходит ответ ― те же пакеты, что и были отправлены. Исходя из этого ответа оценивается работоспособность сервера. Устройство работает нормально, если отправленные пакеты вернулись полностью и быстро.

    4. Немного информации о пунктах 4-5

    DNS(Domain Name System) ― это центральный элемент интернет-системы. DNS соединяет IP-адрес с доменным именем, которое ему соответствует. Благодаря этой системе, нам не нужно запоминать IP-адрес. Достаточно ввести домен в поисковую строку и браузер автоматически преобразует его в IP-адрес. Посмотреть DNS может не только браузер. Любой пользователь может получить информацию о записях при помощи whois или nslookup.

  4. Использование nslookup

    nslookup — это утилита, которая позволяет через командную строку узнать содержимое DNS.
    Утилита поможет узнать IP-адрес и узнать A-запись(С помощью А-записи домен прикрепляется к IP-адресу. Таким образом, А-запись позволяет найти IP) и MX-запись( это информация о домене, указывающая на сервер, который обрабатывает почту. Показывает зарегистрированный почтовый сервер для данного домена) для домена.

  5. Использование whois

    whois — это сетевой протокол, который позволяет узнать основные данные о доменном имени (кто администратор и регистратор, срок регистрации и т.п.).
